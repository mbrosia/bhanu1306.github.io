LinkedIn Employeee Profile Scrapper

Web Scrapper using Python

LinkedIn Employee Profile Scrapper is a tool that is based on Python. It is used to scrap the employee profile for checking the employee profile details on a regular basis.
The tool is a part of a analysis software, in which we tried to predict whether there are chances that the employee is about to leave the company. It a periodic scraping tool which once started, will scrap data on the regular interval of time that we have specified. The input file is a list of employee profile URLs and the input login id and password of the employer. The output is generated in a CSV file after the regular interval. It checks whether the employee has updated his or her Designation, Skills, Number of increased connections, Number of increased connections with the recruiter designation.
It uses Selenium with Python, BeuitifulSoup, Pandas, csv, and other python standard packages. The source code of the application is hosted on Github
